How it works

Smart mode (when enable + disable are configured): Press fires the Enable action when turning ON and the Disable action when turning OFF. Since enable/disable are idempotent, the state self-corrects within 1-2 presses if it drifts.

Toggle mode (toggle only, no enable/disable): Press fires the toggle action and flips the visual state. Hold for ~1 second to re-sync the visual state without firing any binding.

Bindings must be loaded first (use the Settings button to refresh).

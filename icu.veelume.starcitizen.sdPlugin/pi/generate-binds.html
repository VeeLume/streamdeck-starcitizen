Profile Name — name for the generated XML file and SC profile.
Output Path — override the output directory. Leave empty to auto-detect from the active installation.

Press the button to generate conflict-free keyboard bindings for all unbound actions. The generated file can be imported in Star Citizen's keybinding settings.

Existing bindings (both default and user-customized) are respected — the generator only assigns keys to actions that have no keyboard binding.

Note: Generated bindings are written as an XML file for import into Star Citizen's keybinding settings. They are not automatically loaded into the plugin — use the Settings button to reload after importing.

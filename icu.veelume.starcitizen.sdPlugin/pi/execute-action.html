How it works

Short press: executes the primary SC Action when the button is pressed (or released, if hold / double-press is also configured).

Hold: if you hold the button for at least the threshold, the Hold Action fires instead. The primary action does not fire.

Double-press: press the button twice within the window to trigger the Double-Press Action. The primary action fires only if no second press arrives within the window.

Bindings must be loaded first (use the Settings button to refresh).
